C'est non-négociable pour moi: le nom d'un outil c'est son identity. Le menu path c'est "Emil's Tools/[Catégorie]/[Outil]", le class name c'est "EmilsNomDuToolWindow", pis le namespace c'est "EmilsWork.Editor". C'est cohérent, retrouvable dans le projet, pis ça feel pro même si c'est des tools perso.
