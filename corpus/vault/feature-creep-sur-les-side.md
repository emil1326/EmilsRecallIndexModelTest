Ça m'est arrivé avec plusieurs versions du site perso. Tu rajoutes une section, un effet, un easter egg... pis le truc est jamais "assez bon" pour être publié. La solution que j'ai trouvée: se fixer un scope minimal pis ship ça d'abord. Le reste après, si t'en as encore envie.
