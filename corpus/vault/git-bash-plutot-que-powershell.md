Les commandes Git, npm, node, tout ça roule mieux dans Bash que dans PowerShell, genre les pipes fonctionnent comme prévu. PowerShell c'est utile pour des trucs Windows-specifiques mais pour le dev quotidien c'est overkill pis les erreurs sont cryptiques. Git Bash dans Windows Terminal, c'est le sweet spot.
