Format que j'utilise: NomProduit_v1_2_3.unitypackage (underscores parce que les dots peuvent causer des issues sur certains OS). Dans le listing Gumroad, t'updates la description avec un changelog à chaque version. Les buyers reçoivent pas de notification automatique pour les updates à moins que tu leur envoies un email via Gumroad direct.
