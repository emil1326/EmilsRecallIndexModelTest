Genre si t'as utilisé des base meshes, des textures tierces, ou des scripts externes, faut créditer dans le package. C'est aussi là que tu mets tes ToS: pas de redistribution, usage commercial ok ou pas, etc. Un simple fichier txt dans le root du folder suffit, mais l'absence de ça c'est une red flag pour les buyers sérieux.
