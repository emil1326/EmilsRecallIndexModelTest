Même pour un projet Unity ou un script Python, il y a souvent un `package.json` quelque part pour gérer le tooling. Node.js c'est genre la colle entre tous les outils, genre npm scripts, CLI tools, local dev servers, tout passe par là. C'est pas toujours le plus logique mais tsu, le JavaScript ecosystem c'est tentaculaire.
