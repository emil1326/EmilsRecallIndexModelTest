Sans Settings Sync, réinstaller VS Code c'est une heure à réinstaller des extensions pis retrouver ses settings, c'est plate tbh. Le sync se fait via le compte GitHub ou Microsoft, ça prend deux minutes à activer. Le seul truc qui sync pas parfaitement c'est les extensions qui ont des dépendances système, mais c'est rare.
