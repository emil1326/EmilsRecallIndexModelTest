title: Kebab-case obligatoire pour les slugs URL
summary: Les slugs URL utilisent le kebab-case avec des tirets — pas d'underscores ni camelCase — parce que c'est la convention web standard pis ça lit mieux dans une URL.
type: reference

Donc emils-recall-index pas emilsRecallIndex ni emils_recall_index pour les paths. Une règle simple qui évite les incohérences dans les routes pis les liens. Les search engines préfèrent les tirets aussi, c'est un bonus gratuit.
