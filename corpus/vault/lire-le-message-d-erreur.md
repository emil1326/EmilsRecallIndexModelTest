C'est embarrassant mais honnêtement, combien de fois le message d'erreur disait exactement 'cannot find module X at path Y' pis on a passé une heure à chercher ailleurs? Lire le error message complet, lentement, mot par mot, c'est le step zéro. Toujours. Sans exception.
