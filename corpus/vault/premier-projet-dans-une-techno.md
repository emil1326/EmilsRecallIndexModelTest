T'as pas le recul pour faire les bonnes abstractions, tu copies des patterns que tu comprends à moitié, le code est pas clean. C'est inévitable. L'important c'est de le finir quand même — c'est le projet ugly qui te donne le bagage pour faire le deuxième beau. Emil le sait pis il shippe anyway.
