Essayer de maintenir un backlog de tickets pour un side project solo c'est du overhead que j'ai pas envie de gérer. Quand j'ai un use case qui bloque, j'ajoute la feature, pis basta. Les feature flags dans le code servent de pseudo-roadmap visible. Ça marche, pis c'est honest sur comment ça se passe vraiment.
