La doc officielle est écrite par des gens qui connaissent la techno depuis longtemps — ils oublient de préciser les affaires qui sont évidentes pour eux mais pas pour toi. Les gotchas, les edge cases, les patterns à éviter — tout ça c'est dans les forums, les issues GitHub, pis les blog posts de gens qui ont souffert avant toi. La doc c'est juste le starting point.
