Pour des outils qui tournent pendant que je travaille sur ma machine de toute façon, le cloud ça fait pas de sens. La seule downside c'est que le server est offline quand ma machine est off, mais pour un usage solo c'est fine. Si un jour j'ai besoin d'uptime 24/7, je réévaluerai.
