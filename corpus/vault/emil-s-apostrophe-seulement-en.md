'Emil's AAC Applicator' comme titre de fenêtre Unity c'est parfait. EmilsAACApplicator comme class name c'est aussi parfait. Les deux coexistent dans le même fichier sans la moindre friction. Mettre une apostrophe dans un identifier c'est non, pis c'est pas négociable.
