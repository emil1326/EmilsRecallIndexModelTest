Genre j'ai passé 6h sur un bug de shader compile qui venait d'un edge case dans le driver — le fix était un workaround de 3 lignes que j'aurais pu écrire en 20 minutes. Le signal que t'es dans le rabbit hole c'est quand les prochaines heures de debug ont clairement moins de ROI que contourner le problème. Ça prend du recul pour voir ça live tsu.
