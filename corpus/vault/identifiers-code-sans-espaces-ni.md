Le compilateur C# pis le filesystem ont pas de feeling pour les apostrophes, alors Emils dans le code, Emil's dans les labels. C'est la contrainte la plus boring du branding mais elle est réelle. Une fois que t'as internalisé le split, tu fais ça automatiquement.
