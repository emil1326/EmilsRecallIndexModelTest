Le slug c'est pour le routing, les URLs, les paths de fichiers — kebab-case, lowercase, pas d'apostrophe. Le display name c'est pour les headings, les titres de page, les labels UI — espaces, apostrophe, capitalisation normale. Les deux décrivent le même projet pis coexistent sans problème. C'est la même logique que Emils vs Emil's mais pour les projets web.
