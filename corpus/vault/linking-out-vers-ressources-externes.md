C'est la différence entre une note qui dit "les backlinks c'est bien" et une qui cite la source avec le lien — dans 6 mois, le lien te remet dans le vrai contexte. Pour les sources techniques genre documentation ou code, le lien externe est quasi-obligatoire parce que les specs changent. L'URL seule suffit pas, mets aussi le titre et la date d'accès dans le frontmatter parce que les liens rot.
