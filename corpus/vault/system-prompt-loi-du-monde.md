J'ai testé les mêmes instructions en system vs en user turn et la différence est real — le model "croit" ce qui est dans le system comme si c'était les règles fondamentales de son univers. Tes contraintes importantes, tes formats, tes personas, tout ça va dans le system. Le user turn c'est pour la task, pas pour les règles du jeu. Mettre les guidelines critiques juste dans le user message c'est tata.
