Le workflow c'est: build private → join VRChat → test → fix → rebuild public. Ça double le nombre d'uploads mais ça vaut vraiment la peine, genre les crashs ou les shaders broken c'est ben plus honteux quand c'est already public. Le private slot coûte rien de plus.
