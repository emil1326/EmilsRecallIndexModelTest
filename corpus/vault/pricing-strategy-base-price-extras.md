Les buyers aiment avoir le choix — le base package accessible, pis des add-ons pour ceux qui veulent le full experience. C'est aussi plus facile à update indépendamment si tu changes juste les textures alts. Évite les trop de tiers par contre, genre 3 max avant que ça devienne confusant pour tout le monde.
