VCC gère les dépendances, les updates du SDK, pis les packages curated par VRChat en un seul spot propre. Ça évite les situations où t'as une vieille version du SDK qui break les nouveaux features silencieusement. Setup initial c'est un peu de friction mais après c'est smooth — highly recommend si t'as pas encore switché.
