Setup basique: un account test sur un second device ou en switching d'accounts. T'upload en private sur le main, tu switch sur le test account, t'invites le test dans une instance privée, pis tu check comment ton avatar look pour les autres. Crucial pour les shaders, les dynamics, pis le culling.
