Windows Terminal c'est way mieux que les vieilles consoles Windows, rendering GPU-accéléré, support de vrai Unicode, customisation des couleurs par profile. Le setup de base c'est: Git Bash comme default profile, PowerShell disponible en tab si besoin, tout dans la même fenêtre. Le fait de pouvoir splitter les panes est underrated pour watch les logs pendant qu'on code.
