Sur mes projets web perso, j'ai souvent started en JS pur parce que ça just runs, zero config. Le moment où les bugs de type arrivent et que tu passes plus de temps à debugger qu'à build, là TypeScript devient intéressant. Pour un interpréteur ou une classe Image custom, JS pur c'était probablement le bon call.
